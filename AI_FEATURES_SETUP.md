# AI Features Setup Guide

This document explains how to set up and use the AI-powered features in ReviewFlo.

## Prerequisites

You need an Anthropic API key to use the AI features. Get one at: https://console.anthropic.com/

## Environment Setup

1. Add your Anthropic API key to `.env.local`:

```bash
ANTHROPIC_API_KEY=sk-ant-api03-...
```

2. Restart your development server after adding the key.

## Features Implemented

### 1. AI Template Generator (`/api/ai/generate-template`)

Generates personalized review request message templates using Claude AI.

**Parameters:**
- `tone`: `'friendly'` | `'professional'` | `'casual'`
- `channel`: `'sms'` | `'email'`
- `maxWords`: number (optional, defaults to 50 for SMS, 100 for email)
- `includeEmoji`: boolean (optional, defaults based on tone)
- `customContext`: string (optional, additional context about your business)

**Example Request:**
```typescript
const response = await fetch('/api/ai/generate-template', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${session.access_token}`,
  },
  body: JSON.stringify({
    tone: 'friendly',
    channel: 'email',
    customContext: 'We specialize in luxury spa treatments'
  })
});
```

**Response:**
```json
{
  "success": true,
  "template": "Hi [Name]! Thanks for choosing [Business]...",
  "draftId": "uuid",
  "tokensUsed": 150
}
```

### 2. AI Review Response Generator (`/api/ai/generate-response`)

Generates professional responses to customer reviews using Claude AI.

**Parameters:**
- `reviewText`: string (the customer's review)
- `reviewRating`: number (1-5 stars)
- `tone`: `'friendly'` | `'professional'` | `'casual'`
- `maxWords`: number (optional, defaults to 100)
- `specificIssues`: string (optional, for negative reviews - what you're fixing)

**Example Request:**
```typescript
const response = await fetch('/api/ai/generate-response', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${session.access_token}`,
  },
  body: JSON.stringify({
    reviewText: "Great service! The team was very professional.",
    reviewRating: 5,
    tone: 'friendly'
  })
});
```

**Response:**
```json
{
  "success": true,
  "response": "Thank you so much for the kind words!...",
  "responseId": "uuid",
  "tokensUsed": 120
}
```

## Frontend Components

### AITemplateGenerator

Modal component for generating review request templates.

**Usage:**
```tsx
import AITemplateGenerator from '../components/AITemplateGenerator';

<AITemplateGenerator
  open={showModal}
  onClose={() => setShowModal(false)}
  onUseTemplate={(template) => {
    // Do something with the template
    console.log(template);
  }}
/>
```

### AIReviewResponseGenerator

Modal component for generating responses to reviews.

**Usage:**
```tsx
import AIReviewResponseGenerator from '../components/AIReviewResponseGenerator';

<AIReviewResponseGenerator
  open={showModal}
  onClose={() => setShowModal(false)}
  onUseResponse={(response) => {
    // Do something with the response
    console.log(response);
  }}
/>
```

## Database Schema

The AI features use these Supabase tables:

### `ai_review_drafts`
Stores generated review request templates.

**Columns:**
- `id`: uuid (primary key)
- `business_id`: uuid (foreign key to businesses)
- `draft_type`: text ('request_template' | 'review_draft')
- `generated_text`: text
- `prompt_params`: jsonb
- `tokens_used`: integer
- `created_at`: timestamp

### `ai_review_responses`
Stores generated review responses.

**Columns:**
- `id`: uuid (primary key)
- `business_id`: uuid (foreign key to businesses)
- `review_text`: text
- `review_rating`: integer (1-5)
- `generated_response`: text
- `prompt_params`: jsonb
- `tokens_used`: integer
- `was_used`: boolean
- `created_at`: timestamp

## Tier Gating

AI features are gated to the `AI` tier using the `canUseAIFeatures()` function from `lib/tier-permissions.ts`.

Both API endpoints check for AI tier access and return a 403 error with `upgrade: true` if the user doesn't have access.

## Cost Considerations

The AI features use Claude 3.5 Sonnet. Approximate costs:
- Template generation: ~300 tokens per request (~$0.0009)
- Review response: ~400 tokens per request (~$0.0012)

Tokens are tracked in the database for billing/usage analytics.

## Testing

1. Set your business tier to 'ai' in the database
2. Navigate to Settings → AI Features
3. Click "Generate AI Template" or "Generate Review Response"
4. Fill out the form and click generate
5. Review the generated content and click "Use This Template" or "Copy to Clipboard"

## Troubleshooting

**"AI tier required" error:**
- Make sure your business tier is set to 'ai' in the `businesses` table
- Check that `canUseAIFeatures()` returns true for your tier

**"ANTHROPIC_API_KEY not configured" error:**
- Verify the key is in `.env.local`
- Restart your dev server
- Check the key is valid at console.anthropic.com

**Generation fails silently:**
- Check the browser console for errors
- Check server logs for API errors
- Verify your Anthropic API key has credits
