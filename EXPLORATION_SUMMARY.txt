================================================================================
                              KEY FINDINGS
================================================================================

1. PROJECT FRAMEWORK
   - Next.js 16 with Pages Router (NOT App Router)
   - React 19, TypeScript, Tailwind CSS v4
   - Deployed on Vercel
   - Production URL: https://www.usereviewflo.com

2. DATABASE INFRASTRUCTURE
   - Supabase PostgreSQL backend
   - Row-Level Security (RLS) on all tables
   - 15+ tables including AI-ready tables
   - Service role for API backend operations
   - Full migration history in /supabase/migrations/

3. TIER-BASED FEATURE GATING
   - Free Tier: Basic review collection (no backend sending)
   - Pro Tier: Dashboard sending, multi-platform, 3 templates, no branding
   - AI Tier: SMS, CRM integrations, AI features, white-label
   - All tiers cleanly separated via lib/tier-permissions.ts

4. BACKEND SERVICES
   - Email: Resend (37KB comprehensive template library)
   - SMS: Twilio (ready but disabled until May 2026)
   - Auth: Supabase (Email/Password + Google OAuth + Magic Links)
   - Google Integration: Business Profile API + Places API
   - Analytics: PostHog (conversion tracking)
   - Payments: Stripe (prepared, not yet implemented)

5. AI/LLM INTEGRATION STATUS
   ✓ Anthropic SDK installed (@anthropic-ai/sdk v0.79.0)
   ✓ Database tables ready (ai_review_drafts, ai_review_responses)
   ✓ Tier gating infrastructure ready (canUseAIFeatures())
   ✓ UI placeholders in settings (AI Features section, lines 1284-1354)
   ✗ API endpoints NOT YET implemented
   ✗ Claude API integration NOT YET coded
   ✗ Frontend AI components NOT YET built

6. AUTHENTICATION SETUP
   - Supabase Auth with 3 methods: Email/Password, Google OAuth, Magic Links
   - Bearer token validation in API routes
   - CSRF protection via state token
   - Admin role support for /admin dashboard
   - Session management: Supabase tokens + localStorage

7. EMAIL/MESSAGING INFRASTRUCTURE
   - Resend: Beta emails, review request emails, admin notifications
   - All templates use inline CSS, fully styled HTML
   - Tracking tokens for email attribution
   - Reply-to addresses configured
   - SMS: Twilio integration ready (AI tier, disabled until launch)

8. REVIEW COLLECTION FLOW
   - Public route: https://usereviewflo.com/{business-slug}
   - 1-5 star rating → Feedback form (1-4) or Templates (5 stars)
   - Supports white-label branding (AI tier)
   - Configurable review platforms (Google, Facebook, Yelp, Nextdoor)
   - 1-3 template slots (tier-limited)

9. KEY COMPONENTS DEVELOPED
   - ReviewPreview.tsx: Live preview during settings
   - ReviewRequestsList.tsx: Pro/AI request management
   - SendRequestModal.tsx: Send review requests from dashboard
   - GoogleStatsCard.tsx: Pro/AI business stats
   - Settings.tsx: 70KB comprehensive settings page (7 sections)
   - Dashboard.tsx: 25KB main business dashboard

10. SECURITY & BEST PRACTICES
    - RLS prevents cross-business data access
    - Service role restricted to backend only
    - Input validation (email, phone, slug, length)
    - No sensitive data exposed to client
    - Bearer token auth on all protected endpoints
